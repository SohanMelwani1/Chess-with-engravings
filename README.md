# Chess-with-engravings
A chess board game which doesn't require an AMS to print and is easy to assemble piece by piece even for a beginners and a beginner friendly chess board.

The main reason why I decided to design this was for Beginers to learn and play chess more easier.

Instructions:
Print the 'White board piece' and 'white piece' using either white PLA or White PETG
Print the 'Black board piece' and 'Black piece' using either Black PLA or Black PETG
You can print the board with which ever color you want
